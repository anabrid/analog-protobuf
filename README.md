# REDAC Protobuf Protocol

This repository contains the protobuf definitons for anabrid's protocol used for storing circuits, device specifications and, most importantly, communicate with REDAC/LUCIDAC devices.

For more information on how the protocol is used, refer to the [LUCIstack](https://anabrid.github.io/lucistack/lucistack/abstractions/) and [pybrid](https://anabrid.github.io/pybrid-computing/developer-guide/data-and-messaging-protocol/) documentation. Note that more documentation, including comments on the protocol definition, will be added in the near future.
